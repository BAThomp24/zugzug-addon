# ZugZug — ZUGZUG.io Talent Builds In-Game

Stop alt-tabbing to find the right talents. ZugZug brings the top raid and Mythic+ builds from [ZUGZUG.io](https://zugzug.info) directly into your talent frame.

## What It Does

ZugZug adds a build bar to the bottom of your talent window showing the most popular talent builds for your current class and role — sourced from thousands of top-performing WarcraftLogs parses, updated daily.

- **Raid builds** with per-boss recommendations — see which build is best for each encounter
- **Mythic+ builds** with per-dungeon recommendations — know which build dominates each key
- **All roles supported** — DPS, Healer, and Tank builds included
- **One-click import** — click any build to load it into the talent preview, then hit Apply

## Features

- Automatically detects your class, spec, and role
- Builds appear when you open your talent frame — no extra windows to manage
- **Raid difficulty toggle** — switch between Heroic and Mythic builds
- **M+ key level filter** — filter builds by All / 15+ / 18+ / 20+ keys
- **Hover tooltips** show which bosses or dungeons each build is best for
- **Popularity %** and **trend indicators** (NEW / ▲ / ▼) on every build
- Settings saved between sessions

## Installation

1. Download the latest release from [Releases](https://github.com/BAThomp24/zugzug-addon/releases)
2. Extract the `ZugZug` folder into your `World of Warcraft/_retail_/Interface/AddOns/` directory
3. Restart WoW or `/reload`

## Slash Commands

- `/zz` or `/zugzug` — toggle build panel / show help
- `/zz status` — show current class, spec, settings, and data status
- `/zz diff heroic` or `/zz diff mythic` — set raid difficulty
- `/zz key all` or `/zz key 15+` or `/zz key 18+` or `/zz key 20+` — set M+ key level filter

## Data Source

Build data comes from [ZUGZUG.io](https://zugzug.info), which analyzes top WarcraftLogs V2 rankings across all specs, clusters players by talent fingerprint, and identifies the dominant builds for every boss and dungeon. Data is refreshed daily.
